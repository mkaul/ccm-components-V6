Welcome to ccm Pages.

ccm is a client component model. The client ist the browser, which loads web components on demand from any -maybe cross domain- servers. Every web component consists of CSS, JavaScript and JSON files. JavaScript is used to render HTML into div-s. No iFrames are used.

The Clientside Component Model (ccm) is yet another componentization model for the Web. It is based on a framework developed by Andr√© Kless in his master thesis 2015 at the University of Applied Sciences Bonn-Rhein-Sieg introducing JavaScript+CSS-JSON-components which may be loaded cross-domain cross-platform via JSONP and rendered into any div or span in any web page. For introductory tutorials see the Wiki Pages in his GitHub repository
