Welcome to ccm Pages.

ccm is a client side component model. The client ist the browser, which loads web components on demand from any -maybe cross domain- servers. Every web component consists of CSS, JavaScript and JSON files. JavaScript is used to render HTML into div-s. No iFrames are used.
